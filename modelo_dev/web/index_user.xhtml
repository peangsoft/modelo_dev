1. Economato System é um sistema que permite requisitar qualquer mateial de ecritório existente no stock da MDS

2. É importante que o funcionário esteja previamente registado no domínio de email da MDS.

3. Todos os pedidos serão registados e confirmados com um email.

4. Qualquer erro na aplicação deve ser reportado para equipa de suporte da MDS (suporte@mdsinsure.com)

5. É de inteira responsabilidade do funcionário gerir suas requisições (mantenha a aplicação fechada após utilização).
